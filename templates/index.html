    About the System

    SatyaScan leverages state-of-the-art machine learning algorithms to identify potentially false information across multiple languages. The system uses a calibrated Linear SVM model trained on a balanced dataset of 20,000 verified news articles.

    The model combines word-level and character-level TF-IDF features with advanced numeric features including sentiment analysis, writing style metrics, and statistical patterns. This multi-dimensional approach ensures high accuracy and reliability in news verification.
